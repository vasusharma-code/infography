API Documentation

Generate infographics programmatically using our REST API

POST /api/generate

Generate infographics based on provided content. Authentication required using x-api-key header.

Authentication

x-api-key: your-api-key-here

Request Body

Parameter Type Description
prompt string The input prompt to generate infographics
language string Output language (Default: English)
ratio enum Aspect ratio (vertical, square, horizontal, portrait)

Example Request

const response = await fetch('https://app.infography.in/api/generate', {
    method: 'POST',
    headers: {
        "x-api-key": "your-api-key-here",
        "Content-Type": "application/json"
    },
    body: JSON.stringify({
        "prompt": "Generate an infographic from https://example.com/blog",
        "language": "English",
        "ratio": "vertical"
    })
});

Response

{
    "imageUrls": [
        "https://example.com/infographic.png"
    ]
}
